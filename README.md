# Proyecto para el tercer control del CDC de la asignatura IS

## Objetivo
Gestionar un cine
## Breve descripción
Se dispone de una serie de funcionalidades asociadas a él:
- Añadir películas a la cartelera (individualmente o en grupo).
- Eliminar películas de la misma (invividualmente o en grupo).
- Devolver el listado de películas que actualmente dispone el cine.
- ...
  
La solución seguirá el patrón MVC. Por ello, se han creado 3 paquetes para organizar el código en función de este patrón.

Se dispone de una clase GerenteTest encargada de realizar algunas pruebas unitarias a la clase Gerente.

El proyecto tiene naturaleza Gradle
